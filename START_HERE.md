TROUBLESHOOTING:
═════════════════════════════════════════════════════════════

Problem: npm not found
  Solution: Install Node.js from https://nodejs.org

Problem: npm install fails
  Solution: Check internet, retry with: npm install

Problem: npm run build fails
  Solution: rm -rf node_modules && npm install && npm run build

Problem: Extension won't load
  Solution: Make sure you selected dist/ folder (not root)

Problem: Manifest error
  Solution: Check dist/manifest.json exists, reload extension

Problem: Models won't download
  Solution: Check internet connection, check browser console (F12)
